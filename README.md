# Jargon (jargon)
Jargon is a platform for Domain Driven Design for APIs and Enterprise Data Modelling. It provides text-based modelling, template-based API design, real-time validation, version control with breaking change detection, and generation of artifacts including JSON Schema, OpenAPI specifications, and JSON-LD. Jargon supports DevOps-driven collaborative modelling for teams designing and governing domains at enterprise scale.
